# Hotel Reservation System

A Java-based Hotel Reservation System developed as part of the CodeAlpha Java Programming Internship.

## Features

* View available hotel rooms
* Room categorization (Standard, Deluxe, Suite)
* Book hotel rooms
* Cancel reservations
* Payment simulation
* View booking details
* Store booking information using File I/O

## Technologies Used

* Java
* File Handling (File I/O)
* Scanner Class
* Object-Oriented Programming (OOP)

## Room Categories

* Standard
* Deluxe
* Suite

## Functionalities

* Search available rooms
* Make reservations
* Cancel bookings
* View reservation details
* Save booking records
